Ejercicio 1: Condiciones de Existencia y Abscisa de Convergencia

Enunciado: Estudiar si la función f(t) verifica las condiciones suficientes para que exista su transformada de Laplace y, en caso afirmativo, determinar la abscisa de convergencia.

Ejemplo: f(t) = { 0, t < 0; 1, t ≥ 0 }.

Solución: La función es continua a trozos y de orden exponencial. Su transformada existe y la abscisa de convergencia es α = 0.

Ejercicio 2: Cálculo de Transformadas de Laplace

Enunciado: Calcular la transformada de Laplace de las siguientes funciones definidas en ℝ+ y comprobar los resultados con Matlab.

a) f(t) = 1.

Solución: F(s) = ∫₀∞ e–st dt = 1/s, para s > 0.

b) f(t) = eat.

Solución: F(s) = ∫₀∞ e–st eat dt = 1/(s – a), para s > a.

Ejercicio 3: Uso de Funciones de Heaviside

Enunciado: Definir la función f(t) utilizando la función escalón (Heaviside) y hallar su transformada de Laplace.

Ejemplo: f(t) = U(t) – U(t – 3).

Solución: F(s) = 1/s – e–3s/s, para s > 0.

Ejercicio 4: Transformada Inversa de Laplace

Enunciado: Calcular la transformada inversa de Laplace de F(s) = 1/[(s² + 1)(s + 1)].

Solución: Descomponiendo en fracciones parciales, se obtiene:
f(t) = (1/2)[cos(t) – sin(t)]e–t, para t ≥ 0.

Ejercicio 5: Resolución de Ecuaciones Diferenciales

Enunciado: Resolver la ecuación diferencial: y″ – 5y′ + 6y = 0, con condiciones iniciales y(0) = 0 y y′(0) = 1.

Solución: Aplicando la transformada de Laplace se transforma la ecuación en una ecuación algebraica, de donde se obtiene:
y(t) = e2t – e3t, que satisface las condiciones iniciales.
